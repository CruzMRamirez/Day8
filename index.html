Commands to learn

  • cd - change directory
  • ls - list out items in current location
  • mkdir - creates a directory/folder
  • touch - creates a file
  • rm - removes a file
  • rmdir - removes a folder
  • cmd +/- -increases the font size in command line

GitBash Commands

  • git clone ssh - clone a repository/folder
  • Git add - readies files/folders to push to GitHub
  • Git status - checks what files are ready to push
  • Git commit -m “commit message here” - commits files and adds a message to it
  • Git push origin branch-name - pushes/uploads our code to the following branch

Additional Github Commands

  • Git branch - shows all branches that exist
  • Git checkout -b new-branch-name - makes a new branch
  • Git checkout branch-name - switches branches
  • Git pull origin branch-name - pull/downloads our code from a branch
